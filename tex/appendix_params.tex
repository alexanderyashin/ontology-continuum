\section{Appendix C: Parameters, Thresholds, and Tension Functions}
\label{sec:appendix-params}

This appendix defines the parameter sets, threshold families and tension
structures used across all continua \(K_0\)–\(K_{12}\).  
Parameters determine the geometry of admissibility \(\Omega(K)\), the boundary
\(\partial\Omega(K)\), and the location of phase transitions.

\subsection{Parameter Sets}
\label{subsec:appendix-params-sets}

For each continuum:
\[
  \Pi(K)=\{\pi_1,\dots,\pi_n\},
\qquad
\pi_i \in \mathcal{D}_{\pi_i}.
\]

Examples include:
\begin{itemize}
  \item physical: bond probability \(p\), local couplings;
  \item chemical: rate constants, activities;
  \item biological: conductances, channel densities;
  \item cognitive: prediction–error bounds, learning coefficients;
  \item social: communication rates, influence weights;
  \item civilizational: resource throughput, stability parameters;
  \item metatheoretical: logical or coherence coefficients.
\end{itemize}

\subsection{Threshold Functions}
\label{subsec:appendix-params-thresholds}

Threshold functions:
\[
  \Theta : S(K)\to\mathbb{R},
\qquad
  \Theta(s)=\Theta(s;\Pi).
\]

Threshold classes:
\begin{itemize}
  \item dimensional thresholds \(\Theta_{\mathrm{dim}}\),
  \item stability thresholds \(\Theta_{\mathrm{stab}}\),
  \item reaction thresholds \(\Theta_{\mathrm{react}}\),
  \item membrane thresholds \(\Theta_{\mathrm{mem}}\),
  \item excitation thresholds \(\Theta_{\mathrm{spike}}\),
  \item cognitive thresholds \(\Theta_{\mathrm{cog}}\),
  \item cooperation thresholds \(\Theta_{\mathrm{coop}}\),
  \item semantic thresholds \(\Theta_{\mathrm{sem}}\),
  \item cycle thresholds \(\Theta_{\mathrm{cycle}}\).
\end{itemize}

\subsection{Threshold Surfaces}
\label{subsec:appendix-params-surfaces}

The admissible boundary is defined by:
\[
  T(s)=0
  \quad\Longleftrightarrow\quad
  s\in\partial\Omega(K).
\]

General tension landscape:
\[
  T(s)=\max_{\ell}\bigl(T_\ell(s)-\Theta_\ell(s;\Pi)\bigr).
\]

\subsection{Tension Function}
\label{subsec:appendix-params-tension}

Structural tension:
\[
  T:S(K)\to\mathbb{R},
\]
with interpretation:
\[
  T(s)<0 \Rightarrow s\in\Omega(K),\qquad
  T(s)=0 \Rightarrow s\in\partial\Omega(K),\qquad
  T(s)>0 \Rightarrow s\notin\Omega(K).
\]

Tension may depend on potentials, flows and parameters:
\[
  T = f(P,J,\Pi).
\]

\subsection{Composite Thresholds}
\label{subsec:appendix-params-composite}

Multi-threshold admissibility:
\[
  \Omega(K)
  =
  \{s : \max_k ( T_k(s)-\Theta_k(s) ) \le 0 \}.
\]

\subsection{Parameter Evolution}
\label{subsec:appendix-params-evolution}

Parameters evolve according to:
\[
  \frac{d\Pi}{dt}
  =
  W(A,P,\Theta,J,C,T,k,\Pi,t),
\]
capturing:
\begin{itemize}
  \item adaptation (K4–K6),
  \item learning (K6),
  \item biological evolution (K3–K5),
  \item institutional drift (K7),
  \item technological drift (K8),
  \item semantic drift (K11),
  \item metatheoretical adjustment (K9–K12).
\end{itemize}

\subsection{Landscape Interpretation}
\label{subsec:appendix-params-landscape}

The landscape function:
\[
  L(s)=T(s)-\Theta(s;\Pi)
\]
determines admissibility.  
Phase transitions occur when:
\[
  L(s)=0.
\]

\paragraph{K9–K12 (Metatheory and Metasemantics).}
\[
  T = \text{inconsistency or semantic-drift measure}.
\]

\subsection{Compact Summary}
\label{subsec:appendix-params-summary}

This appendix formalises:
\begin{itemize}
  \item parameter sets \(\Pi(K)\),
  \item threshold structures \(\Theta_\ell\),
  \item tension landscape \(T(s)\),
  \item composite admissibility,
  \item and parameter evolution \(W\).
\end{itemize}

These structures determine the geometry of \(\Omega\), the location of
\(\partial\Omega\), all phase transitions, dimension emergence, and collapse
conditions across all levels of the Continuum Framework.
